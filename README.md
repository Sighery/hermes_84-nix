# Hermes v0.11.0 (bytecode 84) Nix build

For some Kindle development, I've had to build this ancient Hermes version
(2022). Doing so required using 3.07 CMake (2017) as well.

Don't ask me how to build this on any other platform. Setting up Nix will be
easier than figuring out ancient CMake and the whole C++ toolchain on whatever
distro you use.
